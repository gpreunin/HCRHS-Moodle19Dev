Style Selection

Selecting a preferred style is a two step process:

  1. You need to tell config.php which stylesheet to get.
  2. You will need to change the file name of the background image files that you want to use.

Selecting Style Sheet

Theme style variations are as follows:


  1. solstice_01

  2. solstice_02

  3. solstice_03

  4. solstice_04

To set up your theme so that it recognizes your preferred style variation:

  1. Open config.php within the solstice folder.
  2. Locate line 9, which, by default, should read:

    $THEME->sheets = array('styles_solstice', 'styles_solstice_01');

  3. Replace what appears in blue above with the numbered theme of your preference.
    • Example:

      $THEME->sheets = array('styles_solstice', 'styles_solstice_03');

  4. Save Changes.

Selecting Background Image Folder

All style sheets grab images from the same location: '/solstice/pix/bg/'. For this reason, you will need to change the name of the "bg" folder for whichever theme variation you prefer to use to simply "bg" after renaming the current "bg" directory to something else ("bg_01" recommended).

I probably made that sound much more complex than it is, so:

  1. Open your "solstice" theme folder, then open "pix".
  2. You will see 5 bg directories:
    1. bg
    2. bg_02
    3. bg_03
    4. bg_04
  3. solstice only pulls background images from the folder called "bg" - so if you have changed to a theme variation other than "solstice_01" in the section above, you will need to also change the corersponding image directory to "bg".
    1. Rename "bg" to "bg_01"
    2. Rename the image folder corresponding to your selected theme variation to "bg"
    3. Done!

Header Customization

Adding a Logo

Moodle's header.html file is set up in two parts:

  1. The first determines what is shown on the front page:

    <div id="header-home" class="clearfix">
    <div class="headermain"><h1><?php echo $heading ?></h1><!--<span class="description"><?php echo $COURSE->summary ?></span>--></div>
    <div class="headermenu"><?php echo $menu ?></div> </div>

  2. The second determines what is shown on all other pages:

    <div id="header" class="clearfix">
    <div class="headermain"><h1><?php echo $heading ?></h1></div> <div class="headermenu"><?php echo $menu ?></div>
    </div>

If you're not interested in "search engine optimization (SEO)," you can simply replace:

<h1><?php echo $heading ?></h1>

with:

<h1><img src="/path/to/your/logo/yourlogo.png" /></h1>

Notice that you can add a logo to the front page, while retaining the "$heading" (course names, etc.) on all of the other pages.

If you're like me and you like search engines to identify your site name using the H1 tag, you can leave the HTML alone, and opt instead to replace the H1 text with an image using CSS (thus allowing search engines to still "read" your site name):

  1. Open "styles_serenity.css".
  2. Search for the string "#header-home .headermain h1".
  3. Replace everything that appears in red with what appears in blue:

    #header-home .headermain h1 {
      font: 1.8em normal;
      background: url(/path/to/your/logo/yourlogo.png) no-repeat;
      text-indent: -9999em;
    }

    • Of course, if you wanted your logo on the course pages as well, you could always add that to the mix:

      #header-home .headermain h1,
      #header .headermain h1 {
        background: url(/path/to/your/logo/yourlogo.png) no-repeat;
        text-indent: -9999em;
      }

      #header .headermain h1 {
        font: 1.4em normal;
      }

  4. Save your changes.

Adding a Tagline

I have added a bit of PHP script that calls up your site description on the front page. By default, this script is commented out. If you want to use it, all you have to do is make the following edit to header.html:

<div id="header-home" class="clearfix">
<div class="headermain"><h1><?php echo $heading ?></h1><!--<span class="description"><?php echo $COURSE->summary ?></span>--></div>
<div class="headermenu"><?php echo $menu ?></div> </div>

Keep in mind that if you want to use this, you will have to keep the description short. Your site description can be edited within your Moodle admin tree at "Front Page > Front Page Settings > Front Page Description"

Changing the Header Size

I have used a header image on both the front page and within the site that is 189px tall. Using CSS, I have set the default height of the front page header to 160px and the header on all other pages to 90px. To change these:

  1. Open "styles_serenity.css".
  2. Search for the string "Header".
  3. Change what follows in red:

    #header-home {
    height: 160px;
    }

    #header {
    height: 90px;
    }

Remember, "#header-home" = front page; "#header" = all other pages

Customizing the Front Page Navigation Menu

For the sake of convenience (and to keep a consistent look and feel throughout the Moodle pages), I have added a navigation menu to header.html. The two default links that appear have been set up to take users "Home" and to your installation's "Courses" page. These links can be replaced with your choice of links, and more links can be added to fill up the bar. To edit the links within the navigation bar:

  1. Open "header.html"
  2. Search for the string "#front-nav"
  3. Change what follows in red to your desired location. Change what follows in blue to your desired text. (To add more links, simply copy and paste the last list item [<li></li>] and make the appropriate changes).

    4. <div id="front_nav" class="clearfix">
      <ul id="nav">
        <li class="page_item first"><a href="<?php echo $CFG->wwwroot ?>">Home</a></li>
        <li class="page_item"><a href="<?php echo $CFG->wwwroot ?>/course/index.php/">Courses</a></li>
      </ul>
    </div>

  4. Save changes.

Keep in mind that identifying a relative path within your Moodle installation is easily accomplished using the above PHP script followed by your desired path. For example: <?php echo $CFG->wwwroot ?>/yourpathhere/anyfile.php/">

Creating Custom Colors

Changing Background Image Colors

For your convenience, I have included the Fireworks PNG files that I used to mock up this theme. To change the background colors, this document will need to be modified and slices will need to be exported to their appropriate location.

It is not recommended to edit using Photoshop because the program is unable to match exact web colors.

To edit the background images:

  1. Open "SOURCE/SOLSTICE_0X_SOURCE.png" using Adobe Fireworks (where "0X" = "01", "02", etc.).
  2. Hide slices and hotspots (available in the "Tools" menu or using by pressing "2" on your keyboard [on a Mac that is]).
  3. Make desired color changes to the document.
  4. Show slices and hotspots (again, in the "Tools menu, or just press "2").
  5. From the menu bar, select File > Export.
  6. Select the file you would like to Export your slices to.
  7. Ensure that the Export settings look similar to the image to the right.
  8. Press "Export"
  9. If everything worked correctly, you should have a file containing 6-8 small images that will be used to make up the background of your theme. Replace the default images with your edited images and you should be good to go.

The same steps can be taken to edit the forum background image (FORUM_BG_SOURCE.png).

Changing the CSS Colors

I have seperated all color related items away from the main CSS document for easy editing. To change the CSS colors of your theme:

  1. Open styles_solstice_0X.css (where "0X" = "01", "02", etc.)
  2. If you know which colors you want to use, enter them into the appropriate spaces. If you are unfamiliar with hex colors, you may want to visit a site such as http://colorcombos.com/ for ideas.

CSS Extras

Just for fun, I have added a CSS document filled with some extra goodies that you may want to use in your site. By default, I have not included this CSS into the core of the theme. So, if you want to use it you will have to "turn it on."

Turning Extras On

To use the CSS Extras included within this theme:

  1. Open config.php and add the following (in blue) to line 9:

    $THEME->sheets = array('styles_solstice','styles_solstice_01','styles_extras');

  2. Save Changes

Notice that the file extension (.css) is not included in this list.

Using CSS Extras

To use the CSS Extras (once turned on), simply enclose any text that you write (in plain text mode) as follows:

<p class="information">Text goes here</p>

<p class="alert">Text goes here</p>

<p class="construction">Text goes here</p>

<p class="download">Text goes here</p>

<code>this is code?</code>

<p class="note">Text goes here</p>

<p class="new">Text goes here</p>

Please let me know if you have any trouble.
